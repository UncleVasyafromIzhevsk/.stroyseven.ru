Тест-план

Объём планируемых работ

В данной работе будет проведен анализ WEB сайта https://stroyseven.ru/

Методы проведения тестирования

  • Для тестирования будут применены методы ручного и автоматизированного тестирования.
  • По критерию доступа к коду будет применён метод "Черного ящика"
  • Цели тестирования будут покрыты функциональным тестированием

Стратегия

  • Проверка регистрации
  • Проверка авторизации
  • Работа с товаром:
    • добавление, удаление в корзину;
    • добавление, удаление в избранное;
    • работа с фильрами и поиском.
  • Проверка переходов, ссылок, кнопок

Тест - дизайн

  • Техника анализа граничных значений
  • Тестирование состояний и переходов
  • Таблица решений

Рабочее окружение

  • Python 3.12
  • behave - библиотека для тестирования сайтов
  • playwright - для взаимодействия с сайтом
  • behave-html-pretty-formatter - заполение отчетности
